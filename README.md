Projeto de analise de dados em Power Bi de uma rede de lojas fictícias.
    
    - Com numero de vendas
    - Devoluções
    - Numeros financeiros
    - Numeros de vendas por filial
    - Localização das lojas
    - Produtos mais, e menos vendidos

Um grafico que toda loja que visa se manter com um alto padrão de qualidade, deveria ter.
